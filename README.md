# opentracker
Shell script for compiling latest [opentracker](https://erdgeist.org/arts/software/opentracker/) from source.

## Usage
```
cd /usr/local/src
git clone https://github.com/Fleshgrinder/opentracker.git
sh opentracker/compile.sh
```
